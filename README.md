# Alerta de Pets Perdidos ZooPaw

Projeto acadêmico desenvolvido na Fatec Praia Grande na disciplina de Programação Orientada a Objetos.

Aplicação web desenvolvida em Java para auxiliar na divulgação de pets perdidos, permitindo cadastro, busca por região e geração de cartazes digitais para compartilhamento em redes sociais.

## Funcionalidades
- Cadastro de pets perdidos  
- Busca por região  
- Geração de cartaz digital  
- Registro de pets encontrados  

## Tecnologias
- Java  
- HTML/CSS  
- Banco de dados relacional
