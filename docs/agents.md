# Agentes de Apoio ao Desenvolvimento

## 1. Ferramentas Utilizadas até o momento

* ChatGPT (OpenAI)
* Gemini (Google)

## 2. Finalidade de Uso

As ferramentas de IA foram utilizadas como suporte ao desenvolvimento do projeto, auxiliando em:

* Estruturação dos requisitos
* Organização da documentação
* Definição de arquitetura do sistema
* Sugestões de implementação
* Esclarecimento de dúvidas técnicas

## 3. Limitações

As decisões finais, implementação e validação do sistema são de responsabilidade do desenvolvedor.

## 4. Observações

O uso de múltiplos agentes de IA teve como objetivo aumentar a produtividade e explorar diferentes abordagens para solução de problemas, mantendo o controle e validação humana em todas as etapas do projeto.
